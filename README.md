# Pokémon GO Counters Infographics

This repository contains infographics for all Pokémon GO Pokémon, built by [Pokémon GO Hub](https://pokemongohub.net/) using their [Pokémon GO Database generator](https://db.pokemongohub.net/).

Use these assets with attribution to Pokémon GO Hub. Each infographic notes the month it was generated so you can track when the data was produced, and showcases normal counters alongside Mega or Primal counter options when available. For deeper customization options, consult the full guides in the [Pokémon GO Database](https://db.pokemongohub.net/).
